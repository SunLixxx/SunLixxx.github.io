P4551 最长异或路径

Problem

题目描述

给定一棵 nnn 个点的带权树，结点下标从 111 开始到 nnn。寻找树中找两个结点，求最长的异或路径。

异或路径指的是指两个结点之间唯一路径上的所有边权的异或。

数据范围

1≤n≤100000;0<u,v≤n;0≤w<2311\le n \le 100000;0 < u,v \le n;0 \le w < 2^{31}1≤n≤100000;0<u,v≤n;0≤w<231。

Solution

设 dxd_xdx​ 表示点 xxx 到根节点的异或路径长度。

答案即为 maxi=1nmaxj=1ndi⊕djmax_{i=1}^nmax_{j=1}^n d_i\oplus d_jmaxi=1n​maxj=1n​di​⊕dj​。

考虑枚举 iii，对 djd_jdj​ 建出 01−trie01-trie01−trie，按位贪心。

具体而言，加入 did_idi​ 的高位为1，则递归 01−trie01-trie01−trie 的左子树，反之递归右子树。
