P4551 最长异或路径

Problem

题目描述

给定一棵 nnn 个点的带权树，结点下标从 111 开始到 nnn。寻找树中找两个结点，求最长的异或路径。

异或路径指的是指两个结点之间唯一路径上的所有边权的异或。

数据范围

1≤n≤100000;0<u,v≤n;0≤w<2311\le n \le 100000;0 < u,v \le n;0 \le w < 2^{31}1≤n≤100000;0<u,v≤n;0≤w<231。

Solution

设 dxd_xdx​ 表示点 xxx 到根节点的异或路径长度。

答案即为 maxi=1nmaxj=1ndi⊕djmax_{i=1}^nmax_{j=1}^n d_i\oplus d_jmaxi=1n​maxj=1n​di​⊕dj​。

考虑枚举 iii，对 djd_jdj​ 建出 01−trie01-trie01−trie，按位贪心。

具体而言，加入 did_idi​ 的高位为1，则递归 01−trie01-trie01−trie 的左子树，反之递归右子树。

时间复杂度 O(nlogn)O(nlogn)O(nlogn)。

Summary

本题使用了 01−trie01-trie01−trie 和按位贪心的思想，算是此类问题的一种思路。

2. 开拓者

Problem

题目描述

有一张包含 nnn 个点与 mmm 条边的无向连通图，每条边都有一个权值 www。

对于一条路径，定义其权值为路径经过的所有边的权值的异或和。路径可以经过重复的点或边。如果经过一条边多次，边权也会被计算多次。

令 F(i,j)F(i,j)F(i,j) 表示从 iii 到 jjj 的所有路径的权值的最大值。

有 qqq 组询问，每组询问给定 lll 和 rrr，求所有满足 l≤i<j≤rl\le i<j\le rl≤i<j≤r 的有序点对 (i,j)(i,j)(i,j) 的 F(i,j)F(i,j)F(i,j) 之和。

数据范围

1≤n,q≤1051\le n,q\le10^51≤n,q≤105，1≤m≤2×1051\le m\le2\times10^51≤m≤2×105，0≤wi<2610\le w_i<2^{61}0≤wi​<261。

Solution

如果问题在树上，显然是好做的。

考虑如何将图上问题转换为树上问题。

跑出原图的 dfsdfsdfs 树，对于原图的非树边，显然就会构成一个简单环。

对于树上的所有环，显然可以随便跑。

因此用一个线性基维护所有环的异或值。

记 max(x)max(x)max(x) 表示 xxx 使用线性基中的元素能异或出的最大值，dxd_xdx​ 表示点 xxx 到根路径的异或值。

答案即为 ∑i=lr−1∑j=i+1rmax(di⊕dj)\sum_{i=l}^{r-1}\sum_{j=i+1}^rmax(d_i\oplus d_j)∑i=lr−1​∑j=i+1r​max(di​⊕dj​)。

考虑如何快速的求值。

记 min(x)min(x)min(x) 表示 xxx 使用线性基中的元素能异或出的最小值。

有这样一个结论 max(x⊕y)=max(x)⊕min(y)max(x\oplus y)=max(x)\oplus min(y)max(x⊕y)=max(x)⊕min(y)。

拆位之后简单计数即可。

复杂度 O(nlogV)O(nlogV)O(nlogV)。

Summary

本题采用了与上个题完全不同的思路，结合了线性基的一些性质，算是此类题目的另外一种思路。
